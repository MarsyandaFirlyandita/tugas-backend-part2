# 🌐 Web Portofolio — Marsyanda Firlyandita

Sebuah website portofolio pribadi yang menampilkan profil, skill, pengalaman, sertifikat, dan galeri karya desain dari Marsyanda Firlyandita.

## 📋 Deskripsi

Website ini dibuat sebagai portofolio digital untuk memperkenalkan diri sebagai seorang **UI/UX Designer & Graphic Designer** dengan pengalaman lebih dari 2 tahun. Website menampilkan berbagai informasi mulai dari profil singkat, keahlian, pengalaman kerja, sertifikat, hingga galeri portofolio desain.

## 🛠️ Teknologi yang Digunakan

- HTML5
- CSS3
- Bootstrap 5
- Google Fonts (Ubuntu)
